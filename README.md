# AI知识库

> 聚焦大语言模型、Agent、Skill与AI热点

## 🌟 项目简介

这是一个面向混合人群的AI知识科普网站，旨在通过系统化的内容组织，帮助不同基础的学习者掌握AI核心知识。

**核心特点：**
- 🔥 热点追踪：每日早7点自动更新AI行业动态
- 📚 系统学习：入门+进阶双层知识体系
- 🎯 聚焦核心：专注LLM、Agent、Skill三大领域
- ✅ 内容准确：所有热点标明来源，科普内容确保准确性

## 🚀 快速开始

### 环境要求

- Node.js 18+
- npm 或 pnpm

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build

# 预览生产版本
npm run docs:preview
```

### 访问地址

- Local: http://localhost:5173/
- Network: http://192.168.64.8:5173/

## 📝 内容规范

### 热点内容

**必须包含：**
- 标题
- 摘要
- 来源链接
- 发布时间
- 难度标签

**示例：**
```markdown
### GPT-5即将发布：多模态能力大幅提升

<span class="difficulty-beginner">热点</span> <span style="color: #999; font-size: 13px;">📖 10分钟</span>

OpenAI计划发布GPT-5，支持更强大的多模态理解和生成能力。

**来源：** [OpenAI官方博客](https://openai.com/blog) | 发布时间：2026-04-07
```

### 科普内容

**必须包含：**
- 学习目标
- 前置知识（进阶内容）
- 核心内容
- 参考资料
- 难度标签

**排序原则：**
- 按由易到难顺序排列
- 每个知识点明确前置知识要求
- 提供清晰的学习路径
