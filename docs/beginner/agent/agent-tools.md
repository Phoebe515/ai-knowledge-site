# 如何使用Agent工具

<span class="difficulty-beginner">入门</span> <span style="color: #999; font-size: 13px;">📖 12分钟阅读</span>

## AutoGPT

### 安装

```bash
git clone https://github.com/Significant-Gravitas/AutoGPT.git
cd AutoGPT
pip install -r requirements.txt
```

### 配置

1. 复制配置文件
2. 填写API密钥
3. 设置参数

### 使用

```bash
python -m autogpt
```

## AgentGPT

### 在线使用

访问 [AgentGPT](https://agentgpt.reworkd.ai/) 直接使用。

### 功能

- 目标设定
- 任务分解
- 自动执行
- 结果展示
