# Agent能做什么

<span class="difficulty-beginner">入门</span> <span style="color: #999; font-size: 13px;">📖 10分钟阅读</span>

## 典型应用场景

### 1. 自动化办公

**任务示例：**
- 自动整理邮件
- 生成会议纪要
- 数据录入和处理
- 报告自动生成

**实际案例：**
每周自动汇总销售数据，生成可视化报告并发送给相关人员。

### 2. 信息检索

**任务示例：**
- 竞品分析
- 市场调研
- 文献检索
- 新闻监控

**实际案例：**
自动搜索行业新闻，筛选重要信息，生成每日简报。

### 3. 决策辅助

**任务示例：**
- 数据分析
- 方案对比
- 风险评估
- 建议生成

**实际案例：**
分析多个投资方案，对比优劣势，给出建议。

### 4. 内容创作

**任务示例：**
- 文章写作
- 社交媒体内容
- 营销文案
- 视频脚本

**实际案例：**
根据主题自动生成系列文章，包括大纲、正文、配图建议。

## 能力边界

### Agent擅长

- 重复性任务
- 信息检索和整理
- 多步骤流程
- 工具调用和集成

### Agent不擅长

- 需要创造力的任务
- 需要情感判断的决策
- 高风险操作
- 需要人工审核的内容
