## ChatGPT Plugins

### 推荐插件

1. **Web Browsing** - 网页浏览
2. **Code Interpreter** - 代码执行
3. **DALL-E** - 图像生成
4. **Zapier** - 自动化集成

## Midjourney

AI图像生成工具，擅长艺术创作。

## Notion AI

集成在Notion中的AI助手，适合文档处理。

## 其他工具

- **Perplexity** - AI搜索引擎
- **Cursor** - AI编程助手
- **Gamma** - AI演示文稿
- **Jasper** - AI写作助手
