# 什么是AI Skill

<span class="difficulty-beginner">入门</span> <span style="color: #999; font-size: 13px;">📖 6分钟阅读</span>

## 定义

AI Skill（AI技能）是指为AI模型添加的特定能力或功能，使其能够完成特定任务。

## 类型

### 1. 工具类

- 代码执行
- 文件操作
- API调用
- 数据处理

### 2. 知识类

- 专业领域知识
- 实时信息获取
- 企业内部知识
- 个性化数据

### 3. 能力类

- 图像生成
- 语音识别
- 视频处理
- 多模态理解

## 与插件的区别

### 插件

- 通常指第三方扩展
- 功能相对固定
- 需要安装配置

### Skill

- 更灵活的能力扩展
- 可以自定义开发
- 与模型深度集成

## 应用场景

- 企业知识库
- 专业工具集成
- 业务流程自动化
- 个性化助手
