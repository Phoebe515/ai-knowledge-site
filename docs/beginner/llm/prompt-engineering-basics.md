# 提示工程基础

<span class="difficulty-beginner">入门</span> <span style="color: #999; font-size: 13px;">📖 12分钟阅读</span>

## 什么是提示工程

提示工程（Prompt Engineering）是指通过设计和优化输入提示，引导AI模型生成更准确、更有用的输出的技术。

简单来说，就是学会如何"问对问题"，让AI给出你想要的答案。

::: tip 核心思想
好的提示词 = 清晰的指令 + 明确的上下文 + 具体的要求
:::

## 提示词的基本结构

### 1. 角色设定

告诉AI扮演什么角色：
```
你是一位资深的Python工程师...
你是一位专业的文案写手...
你是一位耐心的老师...
```

### 2. 任务描述

明确要做什么：
```
请帮我分析这段代码的性能问题...
请帮我写一篇关于AI的文章...
请解释什么是机器学习...
```

### 3. 输出要求

指定输出的格式和要求：
```
要求：
- 800字左右
- 通俗易懂
- 包含3个实例
- 使用表格对比
```

### 4. 示例参考

提供参考示例：
```
例如：
输入：苹果
输出：水果，红色，甜味

输入：西红柿
输出：？
```

## 常用技巧

### 技巧1：明确具体

**❌ 不好的提示：**
> 写一篇文章

**✅ 好的提示：**
> 请写一篇关于"人工智能对未来工作影响"的文章，要求：
> - 1000字左右
> - 适合大学生阅读
> - 包含正反两方面观点
> - 最后给出你的预测

### 技巧2：分步骤引导

复杂任务拆解：

**第一步：**
> 请列出写一篇技术博客的步骤

**第二步：**
> 按照你列出的步骤，帮我写一篇关于Python装饰器的博客

### 技巧3：提供上下文

**❌ 缺少上下文：**
> 如何学习编程？

**✅ 提供上下文：**
> 我是一名市场营销人员，零编程基础，想学习Python做数据分析，每天有1小时学习时间，请给我制定一个学习计划。

### 技巧4：使用示例

**Few-shot Learning（少样本学习）：**
```
请按照以下格式总结文章：

示例：
文章：人工智能正在改变各行各业...
总结：本文讨论了AI对各行各业的影响，主要观点包括...

现在请总结以下文章：
文章：[你的文章内容]
```

### 技巧5：思维链

让AI展示推理过程：
```
请一步步思考并回答：
小明有5个苹果，给了小红2个，又买了3个，请问小明现在有几个苹果？

请展示你的计算过程。
```

## 实战案例

### 案例1：写文章

**提示词：**
```
角色：你是一位科技博主
任务：写一篇关于ChatGPT的介绍文章
要求：
1. 标题吸引人
2. 开头引人入胜
3. 内容包含：什么是ChatGPT、如何使用、应用场景
4. 800字左右
5. 适合普通读者
```

### 案例2：代码解释

**提示词：**
```
请解释以下Python代码的作用，要求：
1. 逐行解释
2. 说明关键函数的作用
3. 给出运行结果
4. 提供改进建议

代码：
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

### 案例3：数据分析

**提示词：**
```
我有一份销售数据（CSV格式），包含：日期、产品、销售额、地区
请帮我：
1. 分析销售趋势
2. 找出最畅销的产品
3. 对比不同地区的表现
4. 给出可视化建议
```

## 高级技巧

### 1. 链式提示

将复杂任务分解为多个简单提示：
```
提示1：列出文章大纲
提示2：根据大纲写第一部分
提示3：根据大纲写第二部分
...
```

### 2. 自我修正

让AI检查和改进自己的输出：
```
请检查你刚才的回答，指出可能的问题并改进。
```

### 3. 角色对话

模拟多方对话：
```
请模拟一场关于"AI是否会取代人类工作"的辩论：
- 正方：AI会创造更多工作机会
- 反方：AI会导致大量失业
每人发言3次，每次100字左右。
```

## 常见错误

### 错误1：提示太模糊

**❌** "帮我写点东西"  
**✅** "帮我写一篇500字的产品介绍"

### 错误2：要求太多

**❌** "写一篇包含所有细节的文章"  
**✅** "写一篇文章，重点介绍3个核心概念"

### 错误3：缺少约束

**❌** "解释机器学习"  
**✅** "用简单的语言解释机器学习，适合高中生理解，包含2个实例"

## 提示词模板

### 学习类

```
我想学习[主题]，我的基础是[当前水平]，目标是[学习目标]，请给我制定一个学习计划。
```

### 写作类

```
请写一篇关于[主题]的[文章类型]，要求：
- 字数：[X]字
- 风格：[正式/轻松/专业]
- 受众：[目标读者]
- 包含：[具体内容]
```

### 分析类

```
请分析[数据/文本/问题]，重点关注：
1. [要点1]
2. [要点2]
3. [要点3]

输出格式：[格式要求]
```
